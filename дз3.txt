GitHub. HW_2
1. На локальном репозитории сделать ветки для:
- Postman - git branch postman
- Jmeter -git branch jmeter
- CheckLists - git branch checklist
- Bag Reports - git branch bagreport
- SQL - git branch sql
- Charles - git branch charles
- Mobile testing - git branch mobiletest

2. Запушить все ветки на внешний репозиторий 
3. В ветке Bag Reports сделать текстовый документ со структурой баг репорта cat > "bag.txt"
4. Запушить структуру багрепорта на внешний репозиторий - git commit git add git push
5. Вмержить ветку Bag Reports в Main  - git merge bagReports
6. Запушить main на внешний репозиторий. git push
7. В ветке CheckLists набросать структуру чек листа. git checkout CheckLists cat > "checklist.txt"

8. Запушить структуру на внешний репозиторий
9. На внешнем репозитории сделать Pull Request ветки CheckLists в main 
10. Синхронизировать Внешнюю и Локальную ветки Main git pull
